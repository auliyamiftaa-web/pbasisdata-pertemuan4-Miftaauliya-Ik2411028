📖 Deskripsi
Repository ini berisi tugas praktikum Basis Data Pertemuan 4.  
Program ini dibuat menggunakan SQL dan dijalankan menggunakan XAMPP / MySQL.

 ⚙️ Tools yang Digunakan
- XAMPP
- MySQL
- phpMyAdmin

 📂 Isi Repository
- File SQL untuk pembuatan database
- Query untuk insert data
- Query untuk menampilkan data

 🚀 Cara Menjalankan
1. Buka XAMPP lalu aktifkan **Apache** dan **MySQL**
2. Buka browser lalu masuk ke:
http://localhost/phpmyadmin
3. Buat database baru
4. Import file `.sql` yang ada di repository ini
5. Jalankan query sesuai kebutuhan

 📝 Tujuan
Tugas ini bertujuan untuk memahami:
- Pembuatan database
- Manipulasi data (CRUD)
- Query SQL dasar\
